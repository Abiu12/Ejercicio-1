Titulo del proyecto: Ejercicio 1-Serie
Proposito del proyecto: Returnar un String con una serie numerica
Version o fecha: 19/04/2020
Como iniciar este proyecto: Ejecutar el main de la clase
Autores: Franco Matias Abiu Mahanaim 19161268
Instrucciones para el usuario:
Para prueba del programa se debe de ejecutar el main de la clase y ingresar un numero.
Solucion: En este programa lo primero que se tuvo que sacar fue la serie para asi poder resolverlo, despues se 
puso diferentes condiciones para los casos, y con operaciones aritmeticas se saco la serie.
